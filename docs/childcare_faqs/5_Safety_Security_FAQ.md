# Childcare Center - Safety & Security FAQ

## Frequently Asked Questions About Safety and Security

### 1. How do you ensure my child's safety?

**Answer:** Multiple layers of security protect your child:

**Physical Security:**
- Locked entry doors (keypad/card access)
- Security cameras (all areas except bathrooms)
- Visitor check-in system
- Fenced outdoor play areas
- Regular safety inspections

**Supervision:**
- Staff always within sight/sound
- Never left alone
- Bathroom monitoring
- Constant headcounts
- Assigned primary caregivers

**Background Checks:**
- All staff FBI background checked
- Child abuse registry clearance
- References verified
- Ongoing training

---

### 2. What is your check-in/check-out procedure?

**Answer:** Strict procedures for child safety:

**Check-In:**
1. Sign in at front desk (digital or paper)
2. List arrival time
3. Note child's health/mood
4. Hand off directly to teacher
5. Never leave child unattended

**Check-Out:**
1. Show photo ID (if requested)
2. Sign out digitally
3. Note pick-up time
4. Teacher releases child to you
5. Receive daily report

**Authorized Pick-Up:**
- Only pre-approved persons can pick up
- Photo ID required for verification
- Emergency password system
- Call ahead for changes

**Unauthorized Attempts:**
- Police called immediately
- Child kept safe inside
- Parent notified

---

### 3. What happens during emergencies?

**Answer:** Comprehensive emergency protocols in place:

**Fire Emergency:**
- Monthly fire drills
- Evacuation to assembly point
- Headcount verification
- Emergency contacts called
- Fire department notified

**Medical Emergency:**
- 911 called immediately
- First aid administered
- Parents notified
- Staff accompanies to hospital if needed
- Incident report filed

**Natural Disaster:**
- Shelter-in-place procedures
- Emergency supplies (72 hours)
- Battery-powered communication
- Regular drills
- Reunification procedures

**Lockdown:**
- Doors locked immediately
- Children moved to safe area
- Police contacted
- Parents updated when safe
- Trained staff protocols

**Emergency Contacts:**
- Keep updated at all times
- List 2-3 backup contacts
- Include work/cell numbers

---

**Answer:** Yes, extensively trained:

**Required Certifications:**
- Pediatric CPR (all staff)

- AED training

**On-Site:**

- AED device
- Emergency medications

### 5. How do you handle food allergies?

**Answer:** Strict allergy protocols:

**Enrollment:**
- Document all allergies
- Provide action plan from doctor
- Photo posted in kitchen/classroom
- Emergency medications stored

**Kitchen Safety:**
- No cross-contamination
- Separate prep areas
- Clear labeling
- Staff trained

**Classroom:**
- Allergy-friendly snacks
- "No sharing food" rule
- Hand washing before/after meals
- Teachers aware of all allergies

**Emergency Response:**
- Epi-pens accessible
- Staff trained to administer
- 911 called immediately
- Parents notified
- Incident documented

**Common Allergen Management:**
- Nut-free facility (if applicable)
- Dairy alternatives available
- Gluten-free options
- Vegan accommodations

### 6. What about COVID-19 and illness prevention?

**Answer:** Enhanced health and safety measures:

**Daily Health Screening:**
- Temperature checks at arrival
- Symptom questionnaire
- Visual health assessment
- Sent home if sick

**Hygiene Practices:**
- Frequent hand washing
- Hand sanitizer available
- Surfaces sanitized regularly
- Toys cleaned daily
- Laundry done on-site

**Ventilation:**
- HVAC filters upgraded
- Windows opened when possible
- Air purifiers in rooms
- CO2 monitoring

**Sick Policy:**
- Stay home if sick
- Fever-free 24 hours
- No vomiting/diarrhea
- Doctor clearance for some illnesses

**COVID-Specific:**
- Follows CDC guidelines
- Isolation area for sick children
- Staff testing protocols
- Communication to families
- Virtual options if needed

### 7. How do you prevent child abduction?

**Answer:** Multiple safeguards prevent unauthorized removal:

**Access Control:**
- Locked doors (code/card access)
- Visitor screening
- No unescorted visitors
- Camera monitoring

**Authorization:**
- Written authorization required
- Photo ID verified
- Password system
- Court orders on file (if applicable)

**Staff Training:**
- Recognize all authorized pick-ups
- Request ID if unsure
- Don't release child if suspicious
- Call police if threatened

**Custody Situations:**
- Court orders strictly followed
- Both parents notified (if required)
- No parent can "prohibit" other without court order
- Neutral in disputes

**Prevention:**
- Never release to unauthorized person
- Confirm unexpected pick-ups
- Monitor parking lot
- Quick lockdown capability

**Building Security:**

### 9. How do you handle bullying or aggressive behavior?

**Answer:** Proactive approach to behavior management:

**Prevention:**
- Social-emotional curriculum
- Kindness emphasized
- Conflict resolution taught
- Positive behavior reinforcement

**When Incidents Occur:**
1. Immediate intervention
2. Separate children if needed
3. Assess for injuries
4. Comfort victim
5. Address aggressor appropriately

**Parent Communication:**
- Both families notified
- Incident report filed
- Follow-up meeting if needed
- Action plan developed

**Consequences:**
- Age-appropriate
- Restorative approach
- Teaching opportunity
- Repeated issues: Meeting with parents

**Serious Aggression:**
- Safety assessment
- Behavior plan required
- Outside evaluation may be needed
- Dismissal possible if unsafe

### 10. Can I see the security camera footage?

**Answer:** Limited access with proper procedures:

**Parent Access:**
- Can request footage of own child
- Requires written request
- Valid reason needed
- Privacy of other children protected

**Approved Reasons:**
- Injury investigation
- Serious incident
- Suspected abuse
- Court order

**Process:**
1. Submit written request
2. Specify date/time/incident
3. Director reviews
4. Approved footage shown in office
5. Cannot copy/photograph

**Recorded Footage:**
- Stored 30 days
- Auto-deleted after
- Backed up offsite
- Secure access only

**Privacy:**
- Other children blurred if possible
- Staff faces may be visible
- Cannot be shared outside

**Denied Reasons:**
- Fishing expedition
- No specific incident
- Invades others' privacy
- Legal/safety concerns
